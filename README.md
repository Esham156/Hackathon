## Pokemon API

# Installation and Usage

- open Git Bash
- `git clone git@github.com:Esham156/Hackathon.git` into a folder of your choice
- open in respective code editor
- `npm i` in terminal
- `npm run dev` in terminal 
- app runs on (http://localhost:5173/)
